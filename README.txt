🌟 Visão Geral do Projeto
Este é um Sistema de Gestão de Projetos desenvolvido em Java com interface gráfica (Swing) 🖥️. Ele foi criado para simular, de forma completa e robusta, o gerenciamento de tarefas, equipes e projetos em um ambiente corporativo. A aplicação permite o cadastro de usuários com perfis específicos, a criação de projetos com regras de negócio claras, e a geração de relatórios detalhados 📈.

O sistema foi projetado para ser intuitivo, com uma interface visual limpa e transições de tela fluidas, proporcionando uma experiência de usuário profissional e funcional.

🎯 Objetivo do Projeto
O principal objetivo é demonstrar um fluxo de trabalho completo de gestão de projetos, desde o cadastro de novos colaboradores até o acompanhamento do progresso das tarefas. O sistema serve como uma prova de conceito funcional, capaz de:

Gerenciar o ciclo de vida de projetos e tarefas. ✅

Controlar o acesso e as permissões de cada usuário. 🔑

Proporcionar uma visão clara do andamento das demandas. 📊

Gerar relatórios para análise e tomada de decisão. 📄

✨ Funcionalidades Entregues
O sistema é completo e inclui as seguintes funcionalidades:

Interfaces Gráficas (GUI) Completas: Telas de cadastro, login, painel de gestão, projetos, tarefas, equipes, usuários e relatórios. 🎨

Fluxo de Navegação Intuitivo: A navegação entre as telas é clara, com botões de "Voltar" e menus que direcionam o usuário para todas as áreas do sistema. ➡️

Gestão de Usuários com Regras de Negócio:

O cadastro de novos usuários valida o formato da senha (com letras maiúsculas e numerais). 🔐

Um login é sugerido automaticamente, seguindo a regra de nome, sobrenome e os últimos três dígitos do CPF. 🆔

O sistema define o perfil do usuário (Administrador, Gerente, Colaborador) com base no cargo inserido. 👨‍💼

Gestão de Projetos e Equipes:

A tela de projetos exibe o status e o progresso em porcentagem. 🚀

Regras de negócio implementadas limitam a alocação de usuários a no máximo 4 projetos simultâneos. 🚫

Um gerente pode alocar membros para as equipes sem limite de pessoas por projeto. 👥

Relatórios e Exportação:

A tela de relatórios possui campos de filtro por projeto, gerente e analista. 🔍

É possível exportar os dados filtrados para um arquivo .csv, que será gerado no mesmo diretório do projeto. 💾

Robustez e Tratamento de Erros:

O código inclui tratamento de exceções para evitar que a aplicação trave, garantindo estabilidade durante a execução. 🛡️

🚀 Passo a Passo para Execução
Para rodar o projeto, siga estas instruções simples:

Baixe o Código: Copie o conteúdo do arquivo SistemaGestaoCompletoFinal.java. 📋

Abra o NetBeans: Crie um novo projeto Java. 💻

Cole o Código: Apague o conteúdo do arquivo principal gerado pelo NetBeans e cole o código completo do nosso sistema. ✔️

Execute o Arquivo: Clique no botão "Run" (ou use o atalho Shift + F6) para compilar e iniciar a aplicação. ▶️

O sistema abrirá na tela inicial, pronta para ser demonstrada.

Possíveis Melhorias Futuras (Opcional)
O projeto pode ser expandido para incluir funcionalidades como:

Persistência de Dados: Salvar os dados em um banco de dados para que as informações não sejam perdidas ao fechar o programa.

Notificações em Tempo Real: Implementar um sistema de notificações visuais na tela para alertar sobre o status das tarefas.

Gráficos de Acompanhamento: Adicionar gráficos e dashboards na tela principal para uma visualização mais dinâmica do progresso dos projetos.

Edição de Tarefas: Implementar a funcionalidade de editar e excluir tarefas de um projeto já criado.
